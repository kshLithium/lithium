# Support

## Questions

For usage questions, workflow ideas, and product feedback, open a GitHub issue.

## Bugs

For reproducible bugs, open a bug report and include:

- your OS version
- whether you ran the packaged app or `npm run dev`
- the selected surface (`Chat`, `Code`, `Paper`, or `Context`)
- exact reproduction steps
- logs, screenshots, or the affected workspace shape if relevant

## Security

Do not open public issues for security vulnerabilities.

Follow the reporting instructions in [SECURITY.md](SECURITY.md).
